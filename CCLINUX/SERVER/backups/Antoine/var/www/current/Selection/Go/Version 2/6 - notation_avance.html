Exercice 6 :

Notation

Voici un début de programme : 
				
					package main

					import "fmt"

					func notation_avancee(note int) {
						// écrivez votre code à partir d'ici
					}

					func main() {
						notation_avancee(0);
						notation_avancee(6);
						notation_avancee(10);
						notation_avancee(11);
						notation_avancee(16);
						notation_avancee(20);
						notation_avancee(21);
					}
Complétez ce programme pour qu'il affiche la phrase suivante, selon la valeur de note :
  • Entre 0 et 5 : "Bouh ! c'est nul."
  • Entre 6 et 9 : "C'est pas encore ça..."
  • 10 : "C'était juste !"
  • Entre 11 et 15 : "Bien joué."
  • Entre 16 et 19 : "Vous avez la maîtrise !"
  • 20 : "C'est un perfeeeect !!"
  • "Va t'acheter des doigts !" sinon
Il faudra un saut de ligne à la fin.
